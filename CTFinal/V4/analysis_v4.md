# V4 结果与分析

## 模型与算法
V4 是针对 V3 假阳性泛滥的一次成功物理门控矫正。
- **强度查验机制（Intensity Gating）**：在提取出嵌套孔洞后，增加了一个基于区域亮度均值的后处理过滤器。
- 只有检出区域的平均亮度低于 40（$\bar{I} < 40$）时，才判定其为真实的液性暗区或血管。

## 结果分析
视频结果中，相较于 V3 那满屏的错误蓝色框，V4 的蓝色内孔洞标注大幅减少。该算法强韧地切除了那些只是因为超声透射不均而稍微变暗的软组织（灰度大约在 50 左右），仅精准锁定了中心亮度极低（代表实质积液）区域。
这个版本的演进说明，单纯的拓扑几何操作（如寻找黑洞）必须叠加超声特定的**回声阻抗-灰度先验门限**才能过滤假阳性。不过，这种静态阈值的过滤并未完全解决超声视频帧极易出现全局亮度漂移的问题，依然会导致部分漏检。

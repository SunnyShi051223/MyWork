# V2 结果与分析

## 模型与算法
V2 是基于传统计算机视觉管道的基础二值化版本。
- **预处理**：应用了 CLAHE (Contrast Limited Adaptive Histogram Equalization) 以解决局部对比度低下问题。
- **阈值分割**：利用 Otsu 算法自适应求取全局最佳阈值进行二值化。
- **轮廓提取**：提取最外层轮廓（`RETR_EXTERNAL`），并利用 Ramer-Douglas-Peucker 算法精简顶点。

## 结果分析
在提供的超声视频运行结果中，V2 版本可以成功通过绿线标定软组织或整体器官的最外侧包络结构。
然而，由于使用了 `RETR_EXTERNAL`，算法仅仅停留在物体“表面”，对于医生往往最关心的**内部空腔、血管横截面以及液性暗区（低回声区）存在“视而不见”的问题**。此外，轮廓边缘受斑点噪声的影响，锯齿化明显，在视觉效果上并不平顺。这证明单纯依靠外部轮廓提取未能反映复杂的体内解剖结构。

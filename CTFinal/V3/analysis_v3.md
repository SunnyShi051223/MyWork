# V3 结果与分析

## 模型与算法
V3 在 V2 的基础上，进行了轮廓拓扑检索的全面升级。
- **层级化轮廓解构**：将 OpenCV 的提取模式更改为 `RETR_TREE`，使得算法可以捕获父级嵌套在子级外部的包含结构。
- 借此，所有具备父节点的轮廓（`Hierarchy[i][3] >= 0`）都被识别为病灶“内部孔洞”。

## 结果分析
相比 V2，视频运行结果显示 V3 成功探测出许多处于软组织深部的封闭空间形状并标为蓝色（内孔洞）。这解决了 V2 无法看到内部结构的问题。
但伴随而来的严重副作用是**高假阳性率**：高频超声换能器打出的声波在掠过浅表组织时会产生声衰减导致深部暗区（声影），并且斑点噪声极大地干扰了图像灰度。V3 会将所有这些因物理特性变暗的正常肌肉也判定为“内部孔洞”，使得超声图像框满了密密麻麻的蓝框。在临床上，这种极大量的噪声干扰（虚警）使得该版本模型无法实际使用。
